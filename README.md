# 验证证书版

解决证书问题
```
# Debian
apt-get install ca-certificates -y
# CentOS
yum install ca-certificates -y
```

安装
```
wget -N "https://raw.githubusercontent.com/ss-daily-backup/Linux-NetSpeed-chiakge/master/tcp-check-cert.sh"
chmod +x tcp-check-cert.sh
./tcp-check-cert.sh
```
